# Example Display

A reference implementation demonstrating how to build a boxBot display.

Use this directory as a template when creating your own displays. Copy it,
rename it, and modify the code to implement your screen layout.

## Structure

```
example_display/
├── __init__.py        # Exports the Display subclass
├── display.yaml       # Metadata: name, description, version, author
├── README.md          # This file
├── assets/            # Fonts, images, icons (optional)
└── requirements.txt   # Additional pip dependencies (optional)
```
