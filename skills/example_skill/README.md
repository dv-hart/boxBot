# Example Skill

A reference implementation demonstrating how to build a boxBot skill.

Use this directory as a template when creating your own skills. Copy it,
rename it, and modify the code to implement your capability.

## Structure

```
example_skill/
├── __init__.py       # Exports the Skill subclass
├── skill.yaml        # Metadata: name, description, version, author
├── README.md         # This file
└── requirements.txt  # Additional pip dependencies (optional)
```
